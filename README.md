# Sales_Dashboard
This project is a Sales &amp; Revenue Analysis Dashboard created using Microsoft Excel. It helps to analyze sales performance, identify top-selling products, and understand revenue trends.

Objectives

Analyze total sales and revenue
Identify top-performing products
Track sales trends over time
Provide interactive analysis using filters
🛠 Tools Used
Microsoft Excel
Pivot Tables
Bar Chart
Line Chart
Filters

📈 Features

Total Sales KPI
Top Products Analysis
Sales Trend Over Time
Interactive Filters

📊 Dataset Details

The dataset includes the following columns:

Date
Product
Category
Quantity
Price
Revenue

🚀 Outcome

This dashboard provides clear insights into sales performance and helps in better decision-making.
